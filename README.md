# hue\_exporter

This is a simple Prometheus exporter for various metrics from a Philips Hue system.

This app does obviously not belong to Philips Lighting.

## Installation

Build requirements:
- Crystal 0.24.2

Build it:

```
shards build
```

## Usage

After building, just run it:

```
./bin/hue_exporter
```
